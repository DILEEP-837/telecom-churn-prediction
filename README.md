# Telecom Customer Churn Prediction

This project uses Machine Learning to predict whether a telecom customer will continue recharging or stop using the service.

## Technologies used
- Python
- Pandas
- Scikit-learn

## Machine Learning Algorithm
- Logistic Regression

## Features used
- Age
- Monthly Recharge
- Tenure
- Recharge Frequency

## Output
Predicts whether a customer will churn or not.
